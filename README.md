# 🚀 JavaScript Estudos - Full Stack Rocketseat

Este repositório contém meus códigos de estudo, exercícios e anotações realizados durante a trilha de JavaScript do curso Full Stack da Rocketseat.

## 📚 Tópicos Estudados

O repositório está organizado por módulos que cobrem:

- **Fundamentos:** Objetos, Arrays, Classes e Métodos de texto/array.
- **JavaScript Moderno:** Desestruturação, Spread/Rest operators e Imutabilidade.
- **Assincronismo:** Promises e Funções Assíncronas.
- **Ferramentas:** Compiladores (Babel), Bundlers (Webpack/Vite) e Modularização.
- **APIs:** Integração e consumo de APIs.

## 🛠️ Tecnologias
- JavaScript (ES6+)
- Node.js
- HTML5 / CSS3
